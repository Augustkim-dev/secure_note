## 기술 고려사항

### 앱스토어 설명 (PRD 참고)

**짧은 설명:**
> 대충 입력, 똑똑하게 정리 - AI 비밀번호 노트

**키워드:**
비밀번호, 패스워드, 계정관리, 보안노트, AI, 자동정리, 암호화

### 개인정보 처리방침 포함 내용
- 수집하는 정보 (없음 - 로컬 저장)
- AI 사용 시 데이터 처리 방식
- Import 시 데이터 전송 및 삭제 정책
- 제3자 제공 (Gemini API)

### iOS 심사 주의사항
- 암호화 앱으로 수출 규정 확인
- 데이터 보안 설명 명확히
- 생체 인증 사용 목적 설명

### Android 심사 주의사항
- 민감한 권한 사용 설명
- 데이터 안전 섹션 작성

### 스크린샷 구성 (5장 권장)
1. 잠금 화면 (보안 강조)
2. 노트북 목록 (정리된 모습)
3. Quick Add (한 줄 입력)
4. 분류 결과 (AI 추천)
5. Import 결과 (마이그레이션)
